Literature Survey

Being productive is essential, but it's temporary. To keep up the productivity, it's crucial to stay motivated. In the technological world, researchers are working hard to develop wearable and non-wearable devices like watches, belts, cameras, etc. to monitor users' behavior and suggest necessary changes that can make their lives more productive. These behaviors can be categorized into four main aspects: sleeping patterns, traveling patterns, changes in emotions, and meal-taking patterns.

Several studies have explored the use of mobile sensors for sleep monitoring and one of them is using the accelerometer and gyroscope to monitor sleep stages with an accuracy of 75.4% [1]. To detect traveling patterns, predicting a user's next location is done by using a single, comprehensive dataset that contains location histories, timestamps, and relevant features [2]. Detecting emotions is done by recognizing facial expressions on cell phones using the MobileNet architecture [3]. Image-based nutrition assessment system can recognize a variety of foods, but it requires a device that supports stereo mode [4].

[1] Cheng, P., Zhang, Y., Wang, T., Wang, H., Yang, F., & Zhou, J. (2019). Sleep stage classification is based on unsupervised feature learning and clustering. IEEE

[2] Wang, R., Chen, F., Chen, Z., Li, T., & Harari, G. (2016). Tracking User Sleep Quality via Smartphones: Possibilities and Challenges. ACM Transactions on Computer-Human Interaction, 23(4), 1-27. https://doi.org/10.1145/2988243

[3] Dong, H., Ding, X., & Qian, W. (2020). Prediction of next location based on user's historical data using machine learning algorithm. In 2020 IEEE 3rd International Conference on Electronics Technology (ICET) (pp. 1567-1571). IEEE

[4] H.M Shahzad, Sohail Masood Bhatti, Awais, Mahmood, Arfan Jaffar, “Hybrid Facial Emotion Recognition Using CNN-Based Features”, Faculty of Computer Science and Information Technology.
