Literature Survey

Being productive is essential, but it's temporary. To keep up the productivity, it's crucial to stay motivated. In the technological world, researchers are working hard to develop wearable and non-wearable devices like watches, belts, cameras, etc. to monitor users' behavior and suggest necessary changes that can make their lives more productive. These behaviors can be categorized into four main aspects: sleeping patterns, traveling patterns, changes in emotions, and meal-taking patterns.

Several studies have explored the use of mobile sensors for sleep monitoring and one of them is using the accelerometer and gyroscope to monitor sleep stages with an accuracy of 75.4% [1]. To detect traveling patterns, predicting a user's next location is done by using a single, comprehensive dataset that contains location histories, timestamps, and relevant features [2]. Detecting emotions is done by recognizing facial expressions on cell phones using the MobileNet architecture [3]. Image-based nutrition assessment system can recognize a variety of foods, but it requires a device that supports stereo mode [4].

[1] Cheng, P., Zhang, Y., Wang, T., Wang, H., Yang, F., & Zhou, J. (2019). Sleep stage classification is based on unsupervised feature learning and clustering. IEEE

[2] Wang, R., Chen, F., Chen, Z., Li, T., & Harari, G. (2016). Tracking User Sleep Quality via Smartphones: Possibilities and Challenges. ACM Transactions on Computer-Human Interaction, 23(4), 1-27. https://doi.org/10.1145/2988243

[3] Dong, H., Ding, X., & Qian, W. (2020). Prediction of next location based on user's historical data using machine learning algorithm. In 2020 IEEE 3rd International Conference on Electronics Technology (ICET) (pp. 1567-1571). IEEE

[4] H.M Shahzad, Sohail Masood Bhatti, Awais, Mahmood, Arfan Jaffar, “Hybrid Facial Emotion Recognition Using CNN-Based Features”, Faculty of Computer Science and Information Technology.

Research Problem & Solution

Research Problem


How to detect locations and corresponding times using APIs?


How to detect the sleeping properties of the user by using mobile phone sensors?


How to predict the daily routine analysis by detecting the user emotions?


How to detect food nutrition by capturing the user’s daily intake food?

Predicting a user's next location accurately is challenging due to factors like limited use of machine learning models, oversight in feature engineering, and neglecting weather and traffic conditions. Key questions include the accuracy of emotion detection models and their effectiveness in optimizing daily routines. Furthermore, our mobile application aims to improve productivity by tracking sleep. And also aims to develop a technology-driven system to predict and improve users' food nutrition for better health.

Proposed Solution

Proposed solutions encompass precise location prediction through mobile APIs, real-time emotion detection via image processing in a mobile app, sleep tracking for productivity improvement, and healthy eating support with personalized meal recommendations. Our proposed solution relies on mobile APIs instead of wearables, prioritizing user comfort. It incorporates feature engineering and multiple machine learning models to enhance location prediction accuracy. In the realm of emotion detection, previous studies have identified emotions but lacked comprehensive daily routine analysis. Our research addresses this gap by exploring how image processing can predict emotions for a holistic daily routine analysis. These technologies enhance daily life, offering improved location accuracy, emotional insights, sleep management, and dietary choices, ultimately contributing to better overall well-being.

Methodology

The proposed human daily routine predictor and analyser consists of 4 main components. They are;

  • 1. Understanding sleeping habits and recommending healthy habits
  • 2. Predicting locations for personalized services
  • 3. Recognizing user's emotions and predicting daily routine analysis
  • 4. Recognizing foods and nutrition

All thses 4 components are integrated into a mobile application developed using the Flutter framework. The system aims to cater to a broad audience by ensuring compatibility with both Apple and Android smartphones. Data collection and processing are performed using various techniques and algorithms, involving machine learning models and deep learning methods.The system employs machine learning algorithms like Random Forest Tree, Decision Tree, and Linear Regression to accurately predict users' sleep and wake times based on sensor data and demographics, offering personalized sleep recommendations. It utilizes the geolocator package to collect location data securely, enabling real-time tracking and location-based services. Emotion recognition utilizes Convolutional Neural Networks to detect emotions from real-time face detection, providing users with emotional analysis reports. Additionally, the system employs CNNs to identify food items from photos, estimate nutritional attributes, and assist users in planning their daily calorie intake.Overall, the system provides a comprehensive approach to monitoring users' sleep patterns, predicting locations, recognizing emotions, and detecting food nutrition. By leveraging machine learning and deep learning techniques, the system offers personalized recommendations and analyses, empowering users to make healthier lifestyle choices. The integration of these components into a user-friendly mobile application enhances the overall user experience and promotes well-being.
